# Cosmos

Design system for sckyzo's projects — React 19 + Tailwind 4 + Storybook 9.

## Status

- **Phase 0** (current) : monorepo + tooling + Storybook scaffolding ready, no components yet.
- **Phase 1** : extraction of ~50 components from
  [rackscope](https://github.com/SckyzO/rackscope) (`EmptyPage.tsx`, `TemplateDefaultPage.tsx`,
  `pages/ui/*.tsx`).

## Packages

- [`@sckyzo/cosmos-theme`](https://github.com/SckyzO/cosmos/tree/main/packages/theme) — Tailwind 4 tokens + CSS vars
- [`@sckyzo/cosmos-react`](https://github.com/SckyzO/cosmos/tree/main/packages/react) — React component library
