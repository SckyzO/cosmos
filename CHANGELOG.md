# Changelog

All notable changes to Cosmos packages are documented here.

This repo uses [release-please](https://github.com/googleapis/release-please) to
automate version bumps and changelog generation. Per-package changelogs live in
each `packages/*/CHANGELOG.md`.

## Conventions

- Versions follow [Semantic Versioning](https://semver.org/).
- Commit prefixes (`feat:`, `fix:`, `chore:`, etc.) drive the changelog and
  the next version bump (`feat` → minor, `fix` → patch, `BREAKING CHANGE:` → major).
